Get started

  API Endpoint
 
    http://localhost:8080/api/

This Hello API gives

To set up and use this API clone this repository and set it up.
$ npm install
$ npm start
Go to the API endpoint to test it out.

Get Hello


 # Here is a curl example
 curl \
 -X GET http://localhost:8080/api/hello?lang=en

To get characters you need to make a GET call to the following url:
http://localhost:8080/api/hello 



 Result example :
 
 {
    "lang": "en",
    "language": "English",
    "hello": "Hello"
 }

QUERY PARAMETERS

Field Type Description
lang String The language code. Codes can be found here

Get all Hellos

This gets all Hello in all the languages. You must make a GET call to the following url: http://localhost:8080/api/all 

                    
# Here is a curl example
curl \
-X GET http://localhost:8080/api/all
